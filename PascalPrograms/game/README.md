# MiniHack

Упрощённый roguelike в духе **NetHack**, написанный на Pascal (Free Pascal Compiler).
Процедурно генерируемые подземелья, монстры, предметы, голод, опыт и уровни персонажа —
всё в одном файле `nethack.pas`, без внешних зависимостей, кроме стандартного юнита `crt`.

## Возможности

- 🗺️ Процедурная генерация уровней: комнаты, соединённые коридорами, лестница вниз
- 👁️ Ограниченная область видимости (туман войны), исследуется по мере движения
- 👹 5 видов монстров (крыса, кобольд, гоблин, зомби, орк) с простым ИИ преследования
- 🎒 Предметы: зелья (лечение), еда (снимает голод), золото, оружие (бонус к атаке)
- ⚔️ Пошаговая боевая система с уроном, зависящим от атаки/защиты
- 📈 Система опыта и уровней персонажа (растут HP, атака, защита)
- 🍗 Механика голода — нужно вовремя есть, иначе персонаж начнёт терять здоровье
- ⬇️ Бесконечный спуск по этажам с растущей сложностью
- 🎮 Несколько схем управления одновременно (стрелки, WASD, numpad, vi-клавиши)

## Требования

- [Free Pascal Compiler (fpc)](https://www.freepascal.org/) версии 3.x
- Терминал, поддерживающий юнит `crt` (Linux, Windows, macOS — везде, где есть FPC)

Установка компилятора на Ubuntu/Debian:

```bash
sudo apt-get install fp-compiler
```

## Сборка и запуск

```bash
fpc nethack.pas
./nethack
```

На Windows после компиляции запускайте `nethack.exe` в обычном `cmd`/`PowerShell`.

> Если у вас нет `fpc`, игру можно скомпилировать в [Free Pascal / Lazarus IDE](https://www.lazarus-ide.org/)
> или в любом онлайн-компиляторе Pascal, поддерживающем юнит `crt`.

## Управление

Работает сразу несколько схем — используйте ту, что удобнее:

| Действие              | Клавиши                                  |
|-----------------------|-------------------------------------------|
| Движение              | Стрелки, `W A S D`, Numpad `1-9`, `h j k l y u b n` |
| Подождать на месте    | `.` или Numpad `5`                        |
| Подобрать предмет     | `G` или `,`                                |
| Выпить зелье          | `V`                                        |
| Поесть                | `E`                                        |
| Показать инвентарь    | `I`                                        |
| Спуститься по лестнице| `>`                                        |
| Помощь по управлению  | `?`                                        |
| Выйти из игры         | `Q` (запросит подтверждение `y/n`)         |

Направления Numpad:

```
7 8 9        ↖ ↑ ↗
4 5 6   =    ←  ·  →
1 2 3        ↙ ↓ ↘
```

## Как играть

1. Перемещайтесь по подземелью символом `@`.
2. Врезайтесь в монстра (`r`, `k`, `g`, `z`, `o`), чтобы атаковать его.
3. Подбирайте предметы: `!` зелье, `%` еда, `$` золото, `)` оружие.
4. Следите за голодом — вовремя ешьте (`E`), иначе начнёте терять HP.
5. Найдите лестницу вниз (`>`) и спускайтесь глубже — сложность растёт с каждым этажом.
6. Убивайте монстров, получайте опыт, растите уровень персонажа — и продержитесь как можно дольше.

Игра заканчивается смертью персонажа (HP или голод достигают нуля) — экран в конце покажет,
до какого этажа вы дошли, сколько золота собрали и сколько ходов сделали.

## Структура кода

Всё содержится в одном файле `nethack.pas`:

| Раздел                          | Назначение                                   |
|----------------------------------|-----------------------------------------------|
| `Генерация уровня`               | `GenerateLevel`, `CarveRoom`, `CarveHTunnel/VTunnel` |
| `Отрисовка`                      | `DrawScreen`, `UpdateVisibility`             |
| `Логика`                         | `TryMove`, `PlayerAttack`, `MonsterAttack`, `MoveMonsters` |
| `Предметы и инвентарь`           | `PickUp`, `QuaffPotion`, `EatFood`           |
| `Игровой цикл`                   | главный `while not GameOver` в конце файла   |

## Известные ограничения

Это учебная/развлекательная реализация, а не полный клон NetHack. Отсутствуют:
классы персонажей, заклинания, идентификация предметов, торговцы, ловушки, множество
рас/монстров и артефактов оригинальной игры. Проект задуман как компактный, читаемый
пример roguelike-механик на Pascal, который легко расширять.
