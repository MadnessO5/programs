# Morse Code Transmitter

A terminal-based Morse code encoder written in Pascal. Type any English word and the program will display and play it as Morse code through your speakers.

```
  +------------------------------------------+
  |         MORSE CODE TRANSMITTER           |
  |                                          |
  |   * = dit  (short)   --- = dah  (long)  |
  +------------------------------------------+

  Enter a word: HELLO

  Code table:
    H  ->  ....
    E  ->  .
    L  ->  .-..
    L  ->  .-..
    O  ->  ---

  Transmitting...

  [H]  * * * *
  [E]  *
  [L]  * --- * *
  [L]  * --- * *
  [O]  --- --- ---

  >>> Transmission complete! <<<
```

## Requirements

- **Free Pascal Compiler (FPC)** — to compile the program
- **sox** — to generate audio tones

Install both on Debian/Ubuntu:

```bash
sudo apt install fpc sox
```

## Build & Run

```bash
fpc morse_encoder.pas
./morse_encoder
```

## How It Works

- Each letter is looked up in a built-in A–Z Morse code table
- `*` represents a **dit** (short tone, 80 ms)
- `---` represents a **dah** (long tone, 240 ms)
- Gaps between signals: 80 ms
- Gaps between letters: 240 ms
- Gaps between words: 500 ms
- Audio is generated by calling `sox play` with a 700 Hz sine wave

## Timing Constants

| Constant | Value | Description |
|----------|-------|-------------|
| `FREQ`   | 700 Hz | Tone frequency |
| `DIT_MS` | 80 ms | Length of a dot |
| `DAH_MS` | 240 ms | Length of a dash |
| `SIG_GAP`| 80 ms | Gap between signals |
| `LET_GAP`| 240 ms | Gap between letters |
| `WRD_GAP`| 500 ms | Gap between words |

You can tweak these values in the source code to change the speed and pitch of the transmission.

## Modules Used

| Module | Purpose |
|--------|---------|
| `Crt` | Terminal colors and cursor control |
| `SysUtils` | `IntToStr`, `UpperCase` |
| `Unix` | `fpSystem` to call sox |
